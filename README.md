## 1. Классификация писателей (базовая)
**Файл:** `l5z1(2).ipynb`

### Описание
Проект по определению авторства текста среди 6 известных писателей. Используется метод "скользящего окна" для разделения длинных произведений на обучающие фрагменты.

* **Авторы:** О. Генри, Рэй Брэдбери, Булгаков, Клиффорд Саймак, Макс Фрай, Стругацкие.
* **Технологии:** `Conv1D`, `Embedding`, `Tokenizer`.
* **Особенности:** Проведение экспериментов с различным размером словаря (`VOCAB_SIZE`) для поиска оптимальной точности.

## 2. Классификация медицинских симптомов
**Файл:** `l5z2(1).ipynb`

### Описание
Нейронная сеть для распознавания заболеваний на основе текстового описания жалоб/симптомов пациента.

* **Классы заболеваний:** Аппендицит, гастрит, гепатит, колит, язва и др. (всего 10 классов).
* **Методология:** Тексты разбиваются на обучающую и тестовую выборки в соотношении 80/20.
* **Архитектура:** Использование слоев `Dense` и `Dropout` для классификации очищенных текстовых данных.

## 3. Классификация 20 классических авторов
**Файл:** `l5z3(1).ipynb`

### Описание
Более сложная задача классификации, включающая 20 русских и зарубежных классиков.

* **Список авторов:** Пушкин, Достоевский, Гоголь, Чехов, Толстой, Шолохов, Пастернак и другие.
* **Модель:** Рекуррентная нейронная сеть на базе `LSTM`.
* **Параметры обработки:** Длина фрагмента — 1000 символов, шаг — 100 символов. Словарь ограничен 20 000 наиболее частотных слов.
